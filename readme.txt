Show Hide Read more button for Fusion Builder. It works well for the Avada theme.

== Description ==

This Plugin generates an extra element for Fusion Builder. It allows you to quickly create a button that shows/hides the content of your website. It helps your website look clean and compact, also allows users to click and see more information if they want. Some advantages of this are: robot defense, confirmation of user intent, page loads faster. It will work for every post, pages and custom post types.

== Features ==

* Work for single page template
* Support custom post types
* Increase user readability 
* Decrease SEO bounce rate 


== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "Show Hide Content for Fusion Builder".
3. Click install.
4. Click activate.
